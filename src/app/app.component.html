ng2-carousel

Description

Ng2-Carousel is a simple carousel that any one can use in angular projects, the carousel can be modified in your own way

Eg:

<ng2-carousel [items]="items" [setting]="setting2"></ng2-carousel>

Items : items can be array of images or http url

Settings : 

      setting = {
        autoPlay: true,
        showArrow: true,
        holdTime: 5000,
        showDots: false,
        animation: Ng2AnimationType.FADE_IN_LEFT
      }

MANUAL CONTROL

Eg:

<ng2-carousel #manualControl [items]="items" [setting]="setting2"></ng2-carousel>

#manualControl : This is 'Reference variable' which is used to control ng2-carousel as a @ViewChild

On button click call the function as:

  @ViewChild('manualControl', {static: true}) manualControl: Ng2CarouselComponent;

  playNext() {
    this.manualControl.playNext()
  }

  playPrev() {
    this.manualControl.playPrev()
  }
